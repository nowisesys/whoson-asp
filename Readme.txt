
README FOR WHOSON ASP.NET
--------------------------------

This is the server-side component of the WhosOn application. Deploy this web 
service application on IIS 6.0/7.5 with ASP.NET 2.0. See Deployment.txt for 
more information.

This web service has been tested with .NET 3.5 and gSOAP (C++). See project
page at: https://it.bmc.uu.se/andlov/proj/whoson/asp/
